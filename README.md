# Loser, Am I? — Official Announcement Page

This repository contains a minimalist web page created to mark the official announcement of the book  
**“Loser, Am I? — Beyond Winning and Losing”** by Pranav Gujar.

The page is designed as a **digital archive and announcement**, not as a promotional or sales website.

---

## Purpose

- To document the announcement of the third book
- To host the cinematic announcement video
- To preserve the beginning of this journey in a clean, timeless format

---

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript  

No frameworks, no libraries, no tracking.

---

## Design Philosophy

- Minimal
- Calm
- Black & white / dark theme
- Focused on content, not interaction
- Inspired by film title cards and archival pages

---

## Structure

- Hero section with book title and announcement date
- Embedded announcement video
- Short contextual text
- Reference to previous works
- Minimal footer

---

## Usage

Open the `index.html` file in a browser or host it on any static web server.
